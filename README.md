# Avatar Friends

A fun React app that lets you search and filter a list of avatars by name.
Built with **Create React App**, fetches avatar data from an API, and uses DiceBear for cool avatar images.

## Live Demo

🌐 **Check it out here:**
[https://huzefa077.github.io/avatarFriends_hooks-Redux/](https://huzefa077.github.io/avatarFriends_hooks-Redux/)

## Features

* Search bar to filter avatars in real-time
* Responsive design (works on mobile & desktop)
* Clean UI with card layout
* Loading state & error handling (if API fails)

## Technologies Used

* HTML, CSS, JavaScript
* React (with hooks)
* Redux (for state management)
* DiceBear API (for avatar images)
* JSONPlaceholder API (for fake user/avatar data)

## How to Run Locally

1. Clone the repo

   ```bash
   git clone https://github.com/Huzefa077/avatarFriends_hooks-Redux.git
   ```

2. Go into the project folder

   ```bash
   cd avatarFriends_hooks-Redux
   ```

3. Install dependencies

   ```bash
   npm install
   ```

4. Start the development server

   ```bash
   npm start
   ```

   → Opens at [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run deploy
```
